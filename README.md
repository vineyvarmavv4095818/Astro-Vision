# 🚀Astro-Vision

A Flask-based web application that displays NASA's Astronomy Picture of the Day (APOD).

## Features

* 🌌 Astronomy Picture of the Day
* 📅 Search by Date
* ⬅️ Previous Day Navigation
* ➡️ Next Day Navigation
* 🎲 Random APOD
* ❤️ Add to Favorites
* ⭐ My Favorites Page
*  🗑 Delete Favorites
* 🔍 Search Space topics
* 🎥 Video Support
* 🌌 Deep Space Gallery

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* NASA APOD API
* SQlite

## Installation

1. Clone the repository
2. Install Flask and Requests

```bash
pip install flask requests
```

3. Run the application

```bash
python app.py
```

4. Open:

```text
http://127.0.0.1:5000
```
